# MON AMI

Application web mobile d'autonomie pour les jeunes de 7 a 20 ans.

## Ouvrir l'application

Ouvrir `index.html` dans un navigateur.

## Publier avec GitHub Pages

1. Creer un depot GitHub.
2. Ajouter tout le contenu de ce dossier a la racine du depot.
3. Dans GitHub, ouvrir `Settings`, puis `Pages`.
4. Choisir `Deploy from a branch`, la branche `main` et le dossier `/root`.
5. Enregistrer. GitHub affichera ensuite l'adresse publique de l'application.

Le dossier `assets` doit rester a cote de `index.html`.
